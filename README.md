# web_engineering_2_fullstack_project
This is a fullstack project at the DHBW Mosbach in web engineering 2

## Ideen

- Workout Tracker mit KI, welche man fragen kann auf Basis der User Daten (Workouts, Gewicht, usw.)


### Workout Tracker mit KI

- Workouts tracken für User Daten
- Persönliche Daten/Präferenz (Gewicht, Höhe, Geschlecht, wie oft du trainieren möchtest, Ziel, Präferenzen, Equipment Zugang) für User Daten
- Statistik
- KI Funktionalitäten:
- Kritisieren von Workouts (Weekly Report - Gut, Schlecht, Verbesserungen (basierend auf Präfernenzen, und vorherigen Daten)
- Workoutgenerierung/Vorschlagen von Übungen
- Fragen beantworten durch KI

KI Integration
API Keys fürs lokale Testen bei Gleen
